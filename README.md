# 🍳 Riri's Kitchen Academy

An AI-powered educational cooking game for Emma (age 4) and Elsie (age 9), featuring Riri — their cool big-sister AI coach.

## 🎮 Play the App

Open `index.html` in a browser, or visit the GitHub Pages URL once enabled.

## ✨ Features

- 🤖 AI-powered challenges generated by Claude (age-appropriate for 4 year olds)
- 🎨 Animated Riri character with idle, talking, celebrating and thinking states
- 🍳 Warm illustrated kitchen background
- 🔊 Text-to-speech voice with female voice picker
- ⭐ Star rewards and level progression system
- 🍕 4 dishes: Chocolate Cake, Yummy Pizza, Rainbow Soup, Super Smoothie
- 📱 Designed for iPad / tablet

## 🛠 Tech

Built entirely in HTML/CSS/JS — no framework, no build step. Just open and play.

Powered by the Anthropic Claude API (`claude-sonnet-4-20250514`).
